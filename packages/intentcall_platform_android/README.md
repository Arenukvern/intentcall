# intentcall_platform_android

Federated Android implementation for
[`intentcall_platform`](https://pub.dev/packages/intentcall_platform).

Kotlin package remains `dev.intentcall.intentcall_platform` to match the
Pigeon-generated bridge. Dart API stays in the umbrella package — this package
is native-only.

App authors depend on `intentcall_platform`; this package is endorsed and
resolved automatically.
