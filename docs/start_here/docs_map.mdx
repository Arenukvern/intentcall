# IntentCall Documentation Map

Index for the IntentCall documentation site and repository.

**IntentCall** lets Dart/Flutter apps define agent-callable actions once in `AgentRegistry`, then project them to MCP, WebMCP, shortcuts, and deep links. This repo owns the reusable registry, adapter, session, and platform-projection contracts; [mcp_flutter](https://github.com/Arenukvern/mcp_flutter) is an early consumer and the packaged Flutter product harness.
